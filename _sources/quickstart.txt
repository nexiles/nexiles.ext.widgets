==========
Quickstart
==========

Really Quick Start
==================

Clone the repo and::

  $ cd nexiles.ext.widgets
  $ virtualenv .
  $ . ./bin/activate
  $ . ./setenv.sh
  $ pip install -r requirements.txt
  $ fab init build

Adding the official nexiles ext package Repo
============================================

To use released packages, do::

  $ sencha package repo add nexiles http://developer.nexiles.com/packages/ext
